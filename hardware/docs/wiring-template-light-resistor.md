## 5. 传感器接线

光敏电阻（LDR）阻值会随光强变化（亮 → 阻值小，暗 → 阻值大），跟一个 **10k 固定电阻** 串联组成**分压（voltage divider）**，中间节点喂到 ESP32-C3 的 ADC 引脚。

固件采用 **「亮 → 高电压」** 的接法：把 LDR 接在 3.3V 那一侧，10k 接在 GND 那一侧。

```
  3.3V ── LDR ──┬── GPIO0 (ADC1_CH0)
                │
              10k
                │
              GND
```

| 元件引脚 | 接到 | 备注 |
|----------|------|------|
| LDR 一端 | 3.3V | 光敏电阻一端拉到电源 |
| LDR 另一端 + 10k 一端 | GPIO 0 | **ADC1_CH0** 模拟输入；中间节点电压随光强变化 |
| 10k 另一端 | GND | 10k 拉到地 |

**注意**：
- GPIO0 在 ESP32-C3 上是 ADC1_CH0，固件用 `analogRead(0)` 读取（默认 12-bit，0–4095），归一化成 0..1 后通过 `WB_CH_LIGHT (=41)` 发出。
- 如果你想反过来「暗 → 高电压」，把 LDR 和 10k 的位置交换即可，固件不需要改（只是语义反过来）。
- LDR 元件没有方向，两脚等价；但**整条分压一定要三段都接齐**：3.3V、ADC、GND 都要稳定连接，否则 ADC 读到的电压不可预期。
- 常见 GL5528 / GL5537 在室内大约 ~10k Ω，正好和这颗 10k 固定电阻匹配，分压中点会落在 ADC 量程的中段，留出向亮 / 向暗的双向余量。如果你的 LDR 在常态下阻值很大或很小，可以把 10k 换成接近常态阻值那一档来居中。
